# Livestudy
## 스터디 앱 실행하는법
- mix deps.get
- mix ecto.setup
- npm install (<- assets 폴더에서 실행)
- mix phx.server

## 폴더 구조
- lib/live/public
- lib/live/project

## 규칙
- 참가자는 lib/live/project 폴더에 만들고자하는 프로젝트 폴더를 만든다.
(예시: lib/live/projectx)
- 개인 프로젝트 폴더 이외에 수정가능한 폴더는 
/lib/live/public 폴더 , router 페이지로 한정한다
- 그 이외의 폴더,파일은 수정/삭제하지 않는다.
